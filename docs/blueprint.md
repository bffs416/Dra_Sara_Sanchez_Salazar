# **App Name**: Sara Salazar Aesthetics

## Core Features:

- Before/After Comparison: Interactive before and after image comparison using a lens to reveal changes after procedures.
- Treatment Catalog: Display a catalog of treatments offered, including hair transplants, hyaluronic acid treatments, and radiofrequency therapy.
- Appointment Scheduling: Allow users to schedule appointments for consultations or treatments directly through the app, by integrating a tool such as a calendar system and messaging service.
- Personalized Recommendations: Use a generative AI tool to provide personalized treatment recommendations based on user-provided photos and desired outcomes.
- Medical Journal: Feature a medical journal or blog with articles on the latest trends and science in aesthetic medicine.

## Style Guidelines:

- Primary color: Deep blue (#183862) for trust and sophistication.
- Background color: Very light gray (#F2F2F2), nearly white.
- Accent color: Muted purple (#7F5589) for subtle contrast and elegance.
- Headline font: 'Playfair' serif for elegance; body font: 'PT Sans' sans-serif for readability.
- Use material-style icons to represent treatments and features. All icons should have a filled style.
- Maintain a clean, modern layout with ample white space to emphasize content and imagery.
